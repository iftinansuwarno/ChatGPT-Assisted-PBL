# Contributing to ChatGPT-Assisted PBL

First off, thank you for considering contributing to this research framework! 

## How to Contribute
1. **Report Bugs:** If you find inaccuracies in the pedagogical logic or technical errors, please open an Issue.
2. **Suggestions:** We welcome suggestions for improving the "Human-in-the-Loop" architecture or adding new subject-specific prompts.
3. **Pull Requests:** 
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Ensure your changes align with the ADDIE process and ethical AI guidelines established in the study.
   - Submit a pull request with a detailed description of your changes.

## Academic Integrity
When contributing new lesson plans or rubrics, ensure they follow the same rigorous standards of scientific validity and reliability (KR-20) as the original study[cite: 1].
